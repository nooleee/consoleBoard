# 콘솔 게시판

> ### 소개
>
## 게시판 시스템 베타 서비스<br>
회원가입 후 게시판에 여러가지 글을 작성할 수 있음<br>
작성한 글은 본인만 수정 및 삭제가 가능하며, 다른 사람의 글은 확인 가능함<br>
회원 탈퇴시에도 글은 삭제되지 않아 확인은 가능<br>
